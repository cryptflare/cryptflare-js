# Security Policy

## Reporting a Vulnerability

Email **security@cryptflare.com**. Include:

- Description of the vulnerability
- Reproduction steps or proof of concept
- Affected versions
- Potential impact

We acknowledge reports within **48 hours** and coordinate disclosure through [GitHub Security Advisories](https://github.com/cryptflare/cryptflare-js/security/advisories).

> Do not open public issues for security reports.

## Supported Versions

The latest minor release on npm receives security patches. Older minors receive critical patches at our discretion.

| Version | Supported |
|---------|-----------|
| latest  | yes       |
| < latest minor | critical patches only |

## Disclosure Timeline

We aim to publish a fix within **90 days** of report. Sooner if exploitation is observed.

## PGP

For encrypted communications, request our public key at security@cryptflare.com.
